# CourseSchedule 轻量课程表

一款轻量化的安卓端大学课程表应用，支持每日/每周课表查看、考试提醒、日历总览。(暂时只支持5天制，后续考虑添加周末课程)

> **当前版本**: v1.7 (2026-06-13)  
> **最低 Android 版本**: 8.0 (API 26)  
> **技术栈**: Kotlin + Jetpack Compose + Room + Hilt

---

## 功能概览

### 今日课程
- 打开应用即显示当天课程列表
- 仅显示待上和正在上的课程，**下课后自动消失**
- 当前正在上的课程高亮显示
- 按上课时间排序，显示"下一节"提示

### 周课表
- 周一至周五 × 第1-N节 网格视图
- 支持左右滑动切换周数（弹簧动画，速度+距离双判定）
- 连堂课合并为一个色块显示
- 切换到非本周时周数标红 + **"本周"快捷按钮**
- 左侧节次栏显示每节上下课时间
- 空格长按可快速添加课程
- 已有课程长按进入编辑/删除

### 日历总览
- 月份网格视图，标注当前周数
- 滑动切换月份，点击日期跳转对应周课表
- 长按日期添加考试/提醒
- 显示考试卡片，点击查看详情

### 考试提醒
- 添加考试科目、日期、时间、地点
- 自定义提前提醒时间（x小时）
- 滚轮式时间选择器，精确到15分钟
- 到达提醒时间自动推送通知
- 考试结束后自动清理提醒

### 学期设置
- 自定义节数（支持滚动调整）
- 逐节设置上课/下课时间
- 设置开学日期和总周数（滚动选择器）
- 保存2-3个自定义学期预设

## 技术架构

`
app/
├── data/
│   ├── db/
│   │   ├── AppDatabase.kt          # Room 数据库 (v3)
│   │   ├── entity/                  # Semester, Course, Schedule, Room, Exam
│   │   └── dao/                     # 数据访问对象
│   └── repository/
│       └── CourseRepository.kt      # 数据仓库
├── ui/
│   ├── screen/
│   │   ├── today/                   # 今日课程页
│   │   ├── week/                    # 周课表页
│   │   └── calendar/               # 日历总览页
│   ├── component/                   # 可复用组件
│   └── theme/                       # Material3 主题
├── util/
│   ├── DateUtils.kt                 # 日期计算工具
│   └── NotificationHelper.kt       # 通知管理
├── worker/
│   ├── CourseReminderWorker.kt     # 课程提醒
│   └── ExamReminderWorker.kt       # 考试提醒
└── MainActivity.kt                  # 入口，HorizontalPager + 底部导航
`

**核心依赖**:
- **Jetpack Compose** — 声明式 UI
- **Room** — 本地数据库持久化
- **Hilt** — 依赖注入
- **WorkManager** — 后台定时提醒

## 构建与运行

1. 使用 Android Studio 打开 CourseSchedule/ 目录
2. 等待 Gradle 同步完成
3. 连接安卓设备或启动模拟器
4. 点击 Run 运行

> **注意**: 项目使用 Java 20 编译，请确保 Android Studio 中配置了 JDK 17+。
